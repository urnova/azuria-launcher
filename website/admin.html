📨 Tickets de support

Les tickets envoyes par les utilisateurs (via le site ou le launcher) arrivent directement dans votre canal Discord via webhook.

Pour configurer : Ajoutez la variable DISCORD_WEBHOOK_URL dans les parametres Vercel de votre projet, avec l URL de votre webhook Discord #support-tickets.

Une fois configure, chaque ticket apparaitra dans Discord avec toutes les informations (pseudo, email, categorie, code erreur, description).

🤖 Eva AI — Conversations

Quand un utilisateur clique sur "Transmettre a l equipe de support" dans le chat Eva, la conversation complete est envoyee dans votre canal Discord.

Variables Vercel requises :
• GROQ_API_KEY — Votre cle API Groq (llama-3.1-8b-instant)
• DISCORD_WEBHOOK_URL — Webhook Discord pour les tickets et escalades
