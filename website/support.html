Codes d'erreur du Launcher

Ces codes apparaissent dans le launcher quand quelque chose ne va pas. Retrouve le tien pour connaître la solution.

AZ-001

Session Microsoft expirée

Votre session Microsoft a expiré. Le launcher n'a plus accès à votre compte Minecraft.

✅ Déconnectez-vous et reconnectez-vous avec votre compte Microsoft dans le launcher.
AZ-002

Java 21 introuvable

Java 21 est requis pour NeoForge. Il n'est pas détecté sur votre système.

✅ Le launcher le télécharge automatiquement. Vérifiez votre connexion et relancez.
AZ-003

Minecraft non possédé

Ce compte Microsoft n'a pas acheté Minecraft Java Edition.

✅ Achetez Minecraft sur minecraft.net ou utilisez le mode Hors-Ligne (Crack).
AZ-004

NeoForge introuvable

L'installateur NeoForge est absent. Le launcher ne peut pas lancer le jeu.

✅ Supprimez le dossier .azuria dans %APPDATA% et relancez le launcher.
AZ-005

Téléchargement échoué

La connexion au serveur de mods a échoué (réseau ou pare-feu).

✅ Vérifiez votre connexion internet. Désactivez temporairement votre antivirus.
AZ-006

Extraction des mods échouée

Impossible d'extraire l'archive des mods. Disque plein ou permissions insuffisantes.

✅ Libérez de l'espace disque et vérifiez les permissions du dossier AppData.
AZ-007

Archive des mods vide

Le téléchargement a réussi mais l'archive est vide. Problème serveur temporaire.

✅ Attendez quelques minutes et réessayez. Si ça persiste, contactez le support.
AZ-999

Erreur inconnue

Une erreur non prévue s'est produite. Consultez les logs pour plus de détails.

✅ Envoyez un ticket de support avec vos logs. Notre équipe vous aidera rapidement.
