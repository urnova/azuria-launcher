Guide d'installation
1
Téléchargez et lancez l'installeur

Cliquez sur le bouton ci-dessus pour télécharger AzuriaSetup.exe.

2
Windows SmartScreen — Ne pas paniquer !

Windows affichera peut-être un écran bleu "Windows a protégé votre ordinateur". Cliquez sur "Informations complémentaires" puis "Exécuter quand même". C'est normal pour un nouveau logiciel non signé.

3
Connectez-vous

Ouvrez le Launcher Azuria. Connectez-vous avec votre compte Microsoft Premium ou créez un compte Hors-ligne directement dans le launcher.

4
Jouez !

Cliquez sur Jouer. Les mods se téléchargent automatiquement. En jeu, tapez /menu pour commencer votre aventure.
